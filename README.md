# AuraEcommerce - Angular 17 Demo Website

This is a frontend-only Angular 17 e-commerce demo website based on the Full Stack Developer Assessment document.

## Project Name

AuraEcommerce

## Tech Stack

- Angular 17
- Standalone Components
- Bootstrap 5
- No Backend
- No Database

## Pages

- `/login`
- `/products`
- `/product/:id`
- `/cart`
- `/checkout`

## Features

- Login page with validation
- Product listing
- Product details page
- Add to cart
- Quantity update
- Remove cart item
- Total amount calculation
- Checkout form with validation
- Responsive Bootstrap UI

## Run Project

Open terminal in project folder and run:

```bash
npm install
npm start
```

Then open:

```text
http://localhost:4200
```

## Important Note

This is a demo frontend project. Backend APIs and MongoDB are not connected.
Cart data is stored in memory only and will reset after page refresh.
